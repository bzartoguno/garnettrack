3.6.4 Text Lab: Enter Simple Text

Freestyle Labs are easier to program and test than simulations because they:

  • Involve very little UI work (just this ugly screen) UI work is HARD.
  • Don't require us to simulate features, we just need to be able to parse those features out of a document. This is required by simulations too.
  • Accomodate repeatable testing, all variants of an acceptable/unacceptable document can be generated and automatically tested. i.e. 5 variants that should pass and 5 variants that should partially pass and 5 variants that should fail can all be generated just once.

To pass this lab:

  1. Download the sample word document.
  2. Open the document

  4. Add the word TestOut anywhere in the document.
  5. Save your changes.
  6. Upload your modified document. It's probably in your downloads folder.
